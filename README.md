# Digital-Humanities-Project

## Project description:
This project is my final Digital Humanities project and it compares the journeys of Odysseus and Aeneas in their respective texts using QGIS mapping technologies.

## Contents of repository:
This repository includes: the final QGIS project, the geopackage file containing all data in layers that has been created for this project, the source data for both texts as .csv files, and the Python script used to scrape the data.

### Structure of repository:
* Input folder containing the cleaned .csv files
* Script folder containing the Python script that was used to scrape the data and to create the .csv files
* Output folder containing the finished QGIS file and the geopackages created for this project
